Useful Git Commands

Git add . = Adds all changes within the file to the git console.

Git commit -m "" = Adds all the above changes to a commit message and using -m commits it to memory.

Git push = Pushes the above added files from git add that are attached to the commit message to the github Repo

Tailwind command

npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
